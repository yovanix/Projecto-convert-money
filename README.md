Este projeto, intitulado "convert-money", é um conversor de moedas dinâmico que permite converter as seguintes moedas:
- Peso argentino (Argentina)

- Real brasileiro (Brasil)

- Libra esterlina (Reino Unido)

- Euro (Europa)

- Bitcoin (moeda digital);

Todo o site foi desenvolvido usando HTML, CSS e JavaScript. Ele também possui interação com o teclado, reconhecendo a tecla Enter após a inserção de um valor e executando a conversão diretamente.
